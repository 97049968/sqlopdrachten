# Sql basis

### Database
Maak gebruik van het sql-bestand idscan.sql en voer hierop onderstaande opdrachten uit.
Theorie kun je vinden op: https://www.edutorial.nl/dbq/introductie/

### CSV bestanden importeren
* Maak een database met de naam "superduper" > klaar
* Start mysql-server en log in met je gebruikersnaam en password > klaar
* Download de databestanden > klaar
* Voer het sql-script uit om de tabellen aan te maken in de database "superduper" > klaar
* Importeer de databestanden > klaar
* Exporteer de volledige database inclusief gegevens naar een .sql bestand (bijv. met mysqladmin) > klaar, gedaan in mysql workbench 'table data import wizard'
